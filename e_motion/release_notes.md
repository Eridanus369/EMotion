# E Motion 0.1.0 发布说明

## 初始版本发布

我们很高兴宣布 E Motion 插件的初始版本 0.1.0 正式发布！这是一款专注于运动编辑和曲线操作的 Blender 动画插件，为动画工作流程提供全面的高级工具集。

## 新功能

### 核心功能
- **驱动变量时间重映射**：直观的时间表达式编辑，支持实时预览和调整
- **曲线编辑工具**：曲线发光效果，增强视觉识别，一键删除空曲线和修改器
- **网格曲线编辑器**：基于网格的精确曲线操作，包含贝塞尔曲线编辑工具
- **洋葱皮功能**：动画参考工具，显示前后帧，增强动画流畅度和一致性
- **高级插值方法**：多种自定义插值选项，支持非线性插值和特殊类型
- **轨迹可视化**：显示动画对象的运动轨迹，增强动画规划能力
- **多语言支持**：支持中文、英文、日文、俄文界面

### 技术特性
- 模块化设计，易于维护和扩展
- 优化的性能，即使在处理复杂动画时
- 清晰的用户界面，位于 Graph Editor 的 N 面板
- 完整的功能集成和测试

## 兼容性

- 支持 Blender 4.5.0 至 4.9.0 版本
- 遵循 Blender 插件开发最佳实践
- 完全兼容 Blender 的动画工作流程

## 安装说明

1. 下载插件的 ZIP 文件
2. 在 Blender 中，进入 Edit > Preferences > Add-ons
3. 点击 "Install..." 并选择下载的 ZIP 文件
4. 启用插件，在 "Animation" 类别中找到 "E Motion"

## 使用方法

- 在 Graph Editor 的 N 面板中找到 "E_Motion" 标签
- 在 "Drivers" 模式下使用时间表达式编辑功能
- 在 "FCurves" 模式下使用曲线工具和发光效果
- 访问网格曲线编辑器进行精确操作
- 使用洋葱皮工具参考前后帧动画
