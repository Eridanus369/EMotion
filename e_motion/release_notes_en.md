# E Motion 0.1.0 Release Notes

## Initial Version Release

We are excited to announce the initial release of E Motion add-on version 0.1.0! This is a Blender animation add-on focused on motion editing and curve manipulation, providing a comprehensive set of advanced tools for animation workflows.

## New Features

### Core Features
- **Driver Variable Time Remapping**: Intuitive time expression editing with real-time preview and adjustment
- **Curve Editing Tools**: Curve glow effect for enhanced visual recognition, one-click deletion of empty curves and modifiers
- **Grid Curve Editor**: Grid-based precise curve manipulation with Bezier curve editing tools
- **Onion Skinning**: Animation reference tool showing previous and next frames, enhancing animation smoothness and consistency
- **Advanced Interpolation Methods**: Multiple custom interpolation options, supporting nonlinear interpolation and special types
- **Trajectory Visualization**: Display animation object motion trajectories, enhancing animation planning capabilities
- **Multi-language Support**: Support for Chinese, English, Japanese, and Russian interfaces

### Technical Features
- Modular design for easy maintenance and extension
- Optimized performance even when handling complex animations
- Clear user interface located in the N panel of the Graph Editor
- Complete feature integration and testing

## Compatibility

- Supports Blender 4.5.0 to 4.9.0 versions
- Follows Blender add-on development best practices
- Fully compatible with Blender's animation workflow

## Installation Instructions

1. Download the add-on ZIP file
2. In Blender, go to Edit > Preferences > Add-ons
3. Click "Install..." and select the downloaded ZIP file
4. Enable the add-on by finding "E Motion" in the "Animation" category

## Usage

- Find the "E_Motion" tab in the N panel of the Graph Editor
- Use time expression editing functionality in "Drivers" mode
- Use curve tools and glow effects in "FCurves" mode
- Access the grid curve editor for precise operations
- Use onion skinning tools to reference previous and next frame animations
