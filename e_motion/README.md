# e_motion

A Blender add-on for motion editing and curve manipulation, providing advanced tools for animation workflow.

## Features

- Advanced curve editing tools
- Grid-based curve editor
- Onion skinning for animation
- Custom interpolation methods
- Trajectory visualization
- FCurve modifiers

## Installation

1. Download the latest release from the repository
2. In Blender, go to Edit > Preferences > Add-ons
3. Click "Install..." and select the downloaded ZIP file
4. Enable the add-on by checking the box next to "Animation: e_motion"

## Usage

- The add-on panel can be found in the Animation workspace
- Use the grid curve editor for precise curve manipulation
- Access onion skinning tools for animation reference
- Apply custom interpolation methods to your animations

## Version

Current version: 0.1.0

## Requirements

- Blender 4.0 or later
